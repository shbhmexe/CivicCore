# 🏛️ CivicCore: The Ultimate Technical & Product Encyclopedia

CivicCore is an advanced AI-native infrastructure for digital governance. This document provides an exhaustive breakdown of every module, algorithm, and integration within the platform.

---

## 💎 1. AI Vision & Reasoning Engine (The Core)
*Automating the "Verification of Reality" to prevent fraudulent reports.*

### **Implementation Pipeline**
1.  **Ingestion & Guardrails**:
    *   **SightEngine SDK**: Every image is parsed for NSFW compliance and metadata analysis.
    *   **Synthetic Check**: Detects if an image was generated by AI (DALL-E, Midjourney) to prevent virtual vandalism.
2.  **Semantic Classification**:
    *   **Model**: BART-large-mnli (Zero-Shot).
    *   **Logic**: Maps the visual scene to 6 primary civic categories: `pothole`, `garbage`, `water logging`, `broken streetlight`, `fallen tree`, and `clean road`.
3.  **High-Fidelity Reasoning (Vision LLM)**:
    *   **Model**: Groq Llama-3.2-11b-vision.
    *   **Prompting**: The system asks the model: *"Describe the environment. Is there a physical infrastructure failure here? Return a short citizen-voice description."*
    *   **Output**: This generates the "AI Preview" text seen in the Smart Report form.

## 🛰️ 2. Geographic Intelligence & Proximity Logic
*Using spatial awareness to ensure data accuracy.*

### **Deduplication Engine (Auto-Merge)**
*   **Haversine Algorithm**: When a user reports an issue, the system queries all active reports within a 50-meter radius.
*   **Similarity Scoring**:
    *   **Text Similarity**: HuggingFace `sentence-transformers` calculate the Cosine Similarity between the new report's description and existing ones.
    *   **Decision**: If Similarity > 0.85 AND Distance < 50m, the report is **Auto-Merged**. The new user's vote is added to the original, preventing map clutter.

### **Live Proximity Verification**
*   **Socket.IO Rooms**: Every report creates a virtual room.
*   **Broadcasting**: Coordinates are broadcasted globally.
*   **Client Filtering**: The frontend calculates real-time distance using the browser's Geolocation API. If a user is within 100m, they get a "Nearby Verification Request" modal.

## 🏢 3. Administrative Governance & Escalation
*Holding authorities accountable through automation.*

### **Departmental Routing**
*   **Mapping**: Pothole -> Roads, Garbage -> Sanitation, etc.
*   **Access Control**: Each department can be isolated, but the **Super Admin** sees a holistic "City Pulse" dashboard.

### **The AI Escalation Bot**
*   **Cron Job**: A `node-cron` worker triggers every 6 hours.
*   **Authority Discovery (Groq AI)**:
    *   **Input**: Latitude, Longitude, Department.
    *   **AI reasoning**: *"Based on these coordinates in Gurgaon, which specific MCD office handles sanitation?"*
    *   **Output**: Returns the precise local authority name.
*   **Email Dispatch**: Generates a high-urgency HTML email via **Nodemailer** with a Google Maps deep-link pinned to the issue.

## 📞 4. Voice & Multimedia Integrations
*Multimodal evidence for undeniable proof.*

### **Retell AI Confirmation Calls**
*   **Trigger**: Reports with "High" or "Critical" severity.
*   **Process**: API registers a web-call; reporter receives an automated AI call to verify details verbally.
*   **Verification**: The call transcript is analyzed for keywords. If verified, the `isConfirmed` flag in the database turns green.

### **Video Verification**
*   **Infrastructure**: `next.config.js` updated to support `bodyParser: { sizeLimit: '100mb' }`.
*   **UX**: Citizens can record "Walk-throughs" for issues like water logging where a single photo isn't enough evidence.

## 🏆 5. Gamification & Community Economy
*Building a self-sustaining ecosystem of civic duty.*

### **Karma Points Algorithm**
*   **Report Creation**: +10 Points.
*   **Community Upvote**: +2 Points per vote.
*   **Issue Verification (Nearby)**: +5 Points.
*   **Successful Resolution**: +25 Points.

### **Redemption Store**
*   Modelled via `Redemption` schema. Citizens can exchange points for "Civic Vouchers" (e.g., Property Tax discounts, local shop coupons).

## 🛠️ 7. Technical Stack Deep-Dive
*   **Framework**: Next.js 15 (App Router, Server Actions).
*   **ORM**: Prisma (Type-safe database interactions).
